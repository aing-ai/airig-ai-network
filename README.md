## Key Concepts

### Self-Healing Network
When a participant leaves, the System AI detects the vacant node and redistributes it from the Global Pool to active participants — no manual intervention, no dead nodes, no capital stagnation.

### Empathy Gates
Slots 6–9 are locked behind mandatory charitable donation thresholds. This mathematically forces the most profit-driven users to become the network's largest philanthropists — or forfeit 40% of their potential earning capacity.

### The Gold Mine (Level 3, Model B)
Hardware/commerce referrals at Level 3 earn **5%** — the highest rate in the system. This rewards deep network cultivation over superficial volume recruitment.

### Sovereign User Control (10 Slots)
- Slots 1–3: AI-assigned (best performers from unlimited pool)
- Slots 4–5: User free choice
- Slots 6–7: Empathy Gate I (1% donation threshold)
- Slots 8–9: Empathy Gate II (10% donation threshold)
- Slot 10: APEX — classified
